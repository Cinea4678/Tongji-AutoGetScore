# Tongji-AutoGetScore
自动从同济大学教务网获取成绩并推送通知。

作者：Cinea （信息大类->测绘学院2021级）

纯自学作品，代码写的很丑很不美观，由于你济的分流政策作者无法在大学获得系统性程序设计教育，敬请谅解。

## 开源声明

本项目是开源的，但是若你有修改/引用本项目的需要，你不可以使用tools.py中的邮件账户和密码。
